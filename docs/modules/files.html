Files Module

Intelligent DXF & DWG Import System

The Files Module is the foundation of efficient workflow in GcodePlanner. It provides fast, accurate, and intelligent importing of DXF and DWG files with automatic optimization, making it ideal for woodworking, metalworking, and ACM fabrication shops.

DXF/DWG File Import Preview

Optimized and Cleaned Geometry

Key Features

  • High-fidelity import of DXF and DWG formats with excellent compatibility
  • Intelligent layer recognition and selective import options
  • Automatic geometry cleanup, optimization, and error correction
  • Support for blocks, XREFs, hatches, and complex entities
  • Fast processing even with very large and detailed drawings
  • Seamless integration with Nesting, Machining, and Sheet Metal modules

Thanks to its speed and precision, this module allows you to move from customer drawings to production-ready files in minutes, saving valuable time and reducing manual corrections.
